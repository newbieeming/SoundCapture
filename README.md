# SoundCapture

一个使用 Kotlin + Jetpack Compose 构建的 Android 录音应用，目标是提供轻量、稳定、可扩展的本地录音体验。

## 特性

- 实时录音与录音列表管理
- 录音参数配置（采样率、编码、声道等）
- 基于 Compose 的现代化 UI
- 使用 Hilt 进行依赖注入
- DataStore 持久化应用设置

## 技术栈

- Kotlin
- Jetpack Compose + Material 3
- Android Architecture Components (ViewModel / Lifecycle)
- Hilt + KSP
- Coroutines
- DataStore
